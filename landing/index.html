What it does

Three things, done for you.

🎬

AI-operated. You direct, it edits.

Type "cut the boring parts, add captions, punch in on the hook" — and watch your timeline change. Every action runs through the editor's own tools, so you get a real, editable project, not a black-box render. Drive it with Claude Code (free) or the in-app chat.

⚡

Raw footage → publish-ready reel.

Drop a 20-minute recording; get a tight, captioned, beat-cut short. Kaestral removes filler words, finds the hook, and captions on the word — automatically. Transcription runs on-device; cuts land on silences and beats.

For developers

MCP-first. One line in.

Kaestral is an MCP server. Point the Claude Code you already have at it and start editing by prompt — no new app to learn.

# 1 · start the editor engine (MCP server)
npx kaestral

# 2 · connect Claude Code
claude mcp add --transport http palmier-pro \
  http://127.0.0.1:19789/mcp
claude

# 3 · just ask
# "get_timeline, cut the silent parts of demo.mp4,
#  add captions, and export it."

Three ways to run it

MCP server (primary) — npx kaestral, connect Claude Code. The fastest path for devs.

Windows installer — double-click; the editor + engine start internally, no terminal.

In-app chat — bring your own Anthropic key and edit from the app window.

Compatibility

The MCP server speaks palmier-pro and the .palmier format — a drop-in for the Palmier ecosystem. Exports to Premiere (XMEML) and Resolve/FCP (FCPXML) too.
